# Maechuri
## 개요
매추리 게임은 "매일 추리 게임"이라는 이름으로 AI로 매일 색 다른 추리 게임을 하는 것을 목표로 합니다.

## 기술 스택
- Frontend: React
- Backend: FastAPI, Kotlin + Webflux + Spring Boot 
- Al Tools: Gemini Cli, Github Copilot
